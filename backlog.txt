================================================================================
                    APP NETO - TREINO & DIETA
                    BACKLOG DE DESENVOLVIMENTO
================================================================================

TECH STACK
--------------------------------------------------------------------------------
Camada          | Ferramenta
----------------|---------------------------------------------------------------
Framework       | Next.js 16 (App Router, TypeScript)
UI              | Tailwind CSS 4 + Lucide Icons
Backend/DB      | Supabase (PostgreSQL + Auth + Storage)
Deploy          | Vercel (hosting + cron jobs)
Dominio         | NameCheap (ex: treinoneto.com)
PWA             | next-pwa (instalavel no celular sem app store)
Auth            | JWT (personal) + session (aluno)


ARQUITETURA DO APP
--------------------------------------------------------------------------------

Personal (admin)  -->  Painel Web (desktop/mobile)
                          |
                          |-- Criar/editar treinos
                          |-- Criar/editar dietas
                          |-- Gerenciar alunos
                          |-- Ver progresso dos alunos

Aluno (client)    -->  PWA Mobile
                          |
                          |-- Ver treino do dia
                          |-- Marcar series feitas
                          |-- Registrar cargas usadas
                          |-- Ver dieta prescrita
                          |-- Ver historico/evolucao

[ ] 2. Modelagem do banco (Supabase)
